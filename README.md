# Menter backend
## <u>_Stack_</u>
- Nodejs
- Typescript
- Express
- MongoDB - Atlas(mongoose)
- Firebase Admin
- Testing Suite: Mochajs & Chai

## <u>_Quickstart_</u>

#### <b>Development:</b>
```
 npm run dev
```
#### <b>Test:</b>
```
 npm run test
```
#### <b>Deploy to heroku:</b>
```
 npm run deploy
```
